# Bank Account Management System 💰

## Project Description
The Bank Account Management System is a simple console-based application developed in C++. It allows users to perform basic banking operations such as depositing money, withdrawing money, and viewing account details. The project demonstrates the use of Object-Oriented Programming concepts in C++.

## Features
- Create a bank account
- Deposit money
- Withdraw money
- Check account details
- Menu-driven interface
- Balance validation during withdrawal

## Technologies Used
- C++
- Object-Oriented Programming (OOP)

## Concepts Covered
- Classes and Objects
- Constructors
- Member Functions
- Conditional Statements
- Loops
- Switch Case

## How to Run

1. Clone or download this repository.
2. Open the project in any C++ IDE or compiler.
3. Compile the source file.

Example:

```bash
g++ BankAccount.cpp -o BankAccount
./BankAccount
```
